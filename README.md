# HR Analytics Dashboard

## Project Overview
Developed an interactive HR Analytics Dashboard using Microsoft Excel to analyze workforce trends and employee insights.

## Features
- Attrition Analysis
- Department-wise HC Reporting
- Diversity Analysis
- Employee Stability Tracking
- Interactive Slicers
- Automated Dashboard Buttons using Macros

## Excel Skills Used
- IF Formula
- VLOOKUP
- Pivot Tables
- Charts
- Slicers
- Conditional Formatting
- Assigned Macros

## Tools Used
- Microsoft Excel
- HR Analytics
- Dashboard Reporting
